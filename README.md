# CyberKaido Smart Shell Tool

A simple Bash automation tool that helps organize files and perform basic system tasks on Linux systems.

---

## Description

CyberKaido Smart Shell Tool is a beginner-friendly Bash script designed to automate repetitive file organization tasks in Linux environments.  
It helps reduce manual sorting by grouping scattered files into structured directories.

---

## Features

- Automatically organizes files into folders
- Helps clean up scattered directories
- Lightweight and fast Bash script
- Easy to modify and expand

---

## Requirements

- Linux OS (Kali Linux recommended) 
- Bash shell
- Basic execution permissions

---

Clone or move the script into your system, then run:

chmod +x Smart.sh
